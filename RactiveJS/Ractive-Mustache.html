1. 变量:{{name}}
  2. 部件:
    false或空不显示原则
    以{{#xxx}}开始, 以{{/xxx}}结束
    模板:
    Shown. {{#person}} Never shown! {{/person}}
    数据:
    { "person": false }
    输出: Shown.
    非空集合显示：
    模板:
    {{#repo}} {{name}} {{/repo}}
    数据:
    { "repo": [ { "name": "resque" }, { "name": "hub" }, { "name": "rip" } ] }
    输出:
    <b>resque</b>
    <b>hub</b>
    <b>rip</b>
    非空变量:
    模板:
    {{#person?}} Hi {{name}}! {{/person?}}
    数据:
    { "person?": { "name": "Jon" } }
    输出:
    Hi Jon!
    反转集合：
    以{{^xxx}}开始， 以{{/xxx}}结束
    反转集合取false或空
