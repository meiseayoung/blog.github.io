数据类型 1.一种基本数据类型 Undefined | Null | Boolean | Number | String ( 基本数据类型保存在栈中，按值访问 ) 2.一种复杂数据类型 Object ( 复杂数据类型保存中堆中，按引用访问 ) 数据类型的检测 基本数据类型检测 typeof something 复杂数据类型检测 something instanceof Object|Array|userDefinedObj 执行环境 执行环境定义了变量或函数有权访问的其它数据，决定了他们各自的行为。每个执行环境都有一个与之关联的变量对象，环境中定义的所有变量和函数都保存在这个对象中。 全局执行环境是最外层的一个执行环境。 （某个执行环境中的所有代码执行完毕后，该环境被销毁）
